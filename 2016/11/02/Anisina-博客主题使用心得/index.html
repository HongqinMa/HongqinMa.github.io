Anisina 博客主题使用心得

Posted on 2016-11-02   |  

发现换了一个自己喜欢博客的风格，兴奋到睡不着觉。趁着热腾腾的心情，写下使用心得。

首先，感谢 Haojen Github 的分享，我很喜欢这种简约范儿。

主题优点

主题呢，不言说，简约，image 用来做标题背景，我的最爱。另外，侧边栏: 头像+签名+ github 等网站头像+友链，清爽无杂质，完美的响应式布局，手机版更是清爽美观。

美中不足

不足呢，有以下几点吧。(声明: 本人强迫症所致)

  • 功能方面:
    • 心里老惦记着那位 next 大神的打赏功能，说实话选择换主题时心里痒痒了一小会儿。
  • 图片链接问题:
    • 七牛云CDN是什么鬼？(原谅我的无知)

那么，聊聊七牛云。早在n久前就应该买个域名，注册个空间，
可惜的是因为个人原因放弃了。那么，大概的大概，七牛云就是介么个玩意了，存储数据，随用随拿，只不过用网址来拿。于是乎，动手注册七牛云(免费)，第一次用，没魄力直接砸$。然后呢，按照文档，一步一步一步走啊，始终没搞清楚http:your-cdn.com到底是哪个网址，后来经过我大胆的设想，就是 image 的外链地址。哦啦，测试走起，只是，试了好多好多次，图片还是默认一片灰。好吧，放弃，终于的终于变聪明了，浏览器下审查元素啊？！于是乎，弄明白了是https捣的鬼，我的七牛外链是http啊。。。没办法，找啊找啊找啊找，theme 源码中愣是没找到怎么改。(⊙_⊙?) ， 于是忍不住@了Haojen，强暴改了 index 页面和博文页面的 image 的 url ，得，下次 hexo new 文章名 ，然后 hexo clean && hexo g 全废。。。告终，剧落。

  • theme Anisins的主银并没有说明怎么创建myworks，稀饭的不得了，那个页面。
  • 最后一个就是图片大小的设置问题，我看了 theme 主银 issue 下的讨论，有人提到了。

而对于多说，又多了一个知识点。简单易用易上手。

写到这里，基本没发现其他问题了。主题可圈可点。愿默默无闻的我和熠熠发光的其他每个人都努力奋斗，共同进步。

写在最后 技术大杂汇

  • github 分布式管理仓库
  • git 命令
  • 查漏补缺 github+hexo博客搭建
    • node 环境
    • git 环境
    • npm 全局安装
    • hexo init
    • npm install
    • hexo server 启动本地服务
    • localhost:4000 查看博客
    • hexo new 博客文名称 创建博文
    • git clone 地址 下载主题
    • 打开根目录下 -config 配置
    • npm install hexo-deployer-git --save
    • hexo deploy --generate
  • 七牛云 CDN
  • 多说 评论等
坚持原创技术分享，您的支持将是我最大的鼓励！
